# 项目介绍
一个朴素的dubbo网关和平台基础组件

## 依赖组件
1. [Spring Cloud Gateway](quickstart/d1.spring-cloud-gateway.md)
2. [Apollo(配置中心)](quickstart/d2.config-center.md)
3. [Kafka(消息队列)](quickstart/d3.kafka-wrapper.md)
4. [ElasticJob(调度)](quickstart/d4.elasticjob-wrapper.md)

此外还依赖mysql和redis
